Polecenia Git

git init - inicjalizacja folderu (repozytorium). W folderze ukryty plik gitowy
git status - status obecnych procesow, zmian
git add <nazwa pliku> - dodaje plik do kolejki oczekiwania
git commit -m "komentarz" - dodaje zmiany, flaga -m dodaje komentarze
git remote add origin .... - dane podeane w github
git push origin master - wysyla do zdalnego repozytorium to co zostalo skomitowane lokalnie
git clone - pobieranie repozytorium do nowego katalogu
git branch <nazwa galezi> - tworzę nową gałąź
git checkout <nazwa galezi> - przechodze na gałąź
git merge <nazwa pliku> - laczenie wersji z galezi
